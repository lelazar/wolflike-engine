# WolfLike Engine

WolfLike Engine is a small educational 2.5D raycasting game engine written in C# using MonoGame.

The goal of this project is to learn how classic first-person engines, such as Wolfenstein 3D-style raycasters, can be built from scratch using a 2D grid map, player movement, collision detection, and raycasting.

## Project Goals

- Build a small custom 2.5D raycasting engine
- Learn C# game architecture
- Implement player movement and collision
- Implement raycasting from scratch
- Render a Wolfenstein-style first-person view
- Add walls, sprites, entities, weapons, and simple gameplay systems over time

## Technology

- C#
- .NET
- MonoGame
- Visual Studio


## Planned Features

- Single raycast debug visualization
- Field-of-view raycasting
- 2.5D wall projection
- Textured walls
- Sprite rendering
- Enemies and pickups
- Weapon system
- HUD
- Map loading from external files
